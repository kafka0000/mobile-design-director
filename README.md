# Mobile Design Director — Agent Skill

> Senior UX/UI Design Director for mobile app design auditing across all platforms.

A cross-platform mobile design skill for AI coding agents. Provides premium design analysis with:
- **Three-tier proposals** (Safe / Balanced / Avant-Garde)
- **Aesthetic formulas** (spacing, typography, glassmorphism, color psychology)
- **Motion physics** (spring dynamics, haptics, choreography)
- **Platform guidelines** (Apple HIG, Material Design 3, accessibility)

Covers **SwiftUI**, **Flutter**, **Jetpack Compose**, and **React Native**.

## Install

```bash
# Install to all detected coding agents
npx skills add kafka000/mobile-design-director

# Install to specific agents
npx skills add kafka000/mobile-design-director -a claude-code -a cursor -a antigravity

# Global install
npx skills add kafka000/mobile-design-director -g
```

Supports 35+ tools: Antigravity, Claude Code, Cursor, Codex, Windsurf, Cline, Trae, OpenCode, and more.

```
mobile-design-director/

    ├── aesthetic-formulas.md      # Spacing, typography, depth, color
    ├── motion-physics.md          # Springs, haptics, choreography, transitions
    └── platform-guidelines.md    # HIG, Material 3, cross-platform patterns
```

## What It Does

When activated, the skill guides the AI agent to:
1. **Identify** target platform(s) and analyze the design context
2. **Propose** three design directions with dev cost and usability tradeoffs
3. **Recommend** a specific option with design psychology rationale
4. **Provide** platform-specific implementation hints
